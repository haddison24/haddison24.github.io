Lab 3 - Using BootStrap in HTML

Introduction to BootStrap

Responsive web design helps websites look good on computers, tablets, and phones. Bootstrap helps developers build websites that adjust automatically to different screen sizes.

Things I Learned

  • Bootstrap makes websites responsive and adds responsive images. It also adds buttons for users to clickand many built-in design features that help developers create websites faster. With Bootstrap, pages can automatically adjust to different screen sizes such as computers, tablets, and mobile phones, making the website easier to use for everyone.
